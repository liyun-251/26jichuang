# TP3057 引脚功能说明

> 数据来源：TP3057N-NOPB 规格书 (National Semiconductor)
> 封装：16-Pin DIP/SOIC

---

## 引脚总表

> ATE 通道映射：`FSR-1; DR-2; BCLKR-3; PDN-4; BCLKX-44; MCLKX-45; DX-46; FSX-47; TSX-48`

| 引脚 | 信号         |   类型   |   ATE通道   | 功能简述                                    |
| :--: | ------------ | :------: | :----------: | ------------------------------------------- |
|  1  | VBB          |   电源   |      —      | −5V 负电源，由 DPS2 供电                   |
|  2  | GNDA         |   电源   |      —      | 模拟地（所有信号参考点）                    |
|  3  | VFRO         | 模拟输出 |      —      | 接收端重建音频输出，可驱动 600Ω 至 +7.2dBm |
|  4  | VCC          |   电源   |      —      | +5V 正电源，由 DPS1 供电                    |
|  5  | FSR          | 数字输入 | **1** | 接收帧同步脉冲 8kHz                         |
|  6  | DR           | 数字输入 | **2** | 接收 PCM 数据输入，在 BCLKR 下降沿锁存      |
|  7  | BCLKR/CLKSEL | 数字输入 | **3** | 接收位时钟 (64kHz~2.048MHz)                 |
|  8  | MCLKR/PDN    | 数字输入 | **4** | 接收主时钟 / 掉电控制 (高电平=掉电)         |
|  9  | MCLKX        | 数字输入 | **45** | 发送主时钟，A-law 需 2.048MHz               |
|  10  | BCLKX        | 数字输入 | **44** | 发送位时钟，将 PCM 数据移出 DX              |
|  11  | DX           | 数字输出 | **46** | 发送 PCM 数据输出，**三态**           |
|  12  | FSX         | 数字输入 | **47** | 发送帧同步脉冲 8kHz                         |
|  13  | TSX          | 数字输出 | **48** | 发送时隙指示，**开漏输出**            |
|  14  | GSX          | 模拟输出 |      —      | 发送输入放大器输出端                        |
|  15  | VFXI−       | 模拟输入 |      —      | 发送放大器反相输入端                        |
|  16  | VFXI+        | 模拟输入 |      —      | 发送放大器同相输入端                        |

---

## 按类型分组

### 电源引脚 (4)

| 引脚 | 信号 | 典型值              | 说明                     |
| :--: | ---- | ------------------- | ------------------------ |
|  1  | VBB  | **−5V ±5%** | 负电源                   |
|  2  | GNDA | **0V**        | 模拟地，所有信号以此为准 |
|  4  | VCC  | **+5V ±5%**  | 正电源                   |

### 数字输入 (8)

| 引脚 | 信号      | 时钟域 |   ATE通道   | 说明                                                    |
| :--: | --------- | :----: | :----------: | ------------------------------------------------------- |
|  5  | FSR       |  接收  | **1** | 8kHz 帧同步。FSR 为高期间，BCLKR 的下降沿将 DR 数据移入 |
|  6  | DR        |  接收  | **2** | PCM 数据输入。在 BCLKR 下降沿锁存，MSB 先传             |
|  7  | BCLKR     |  接收  | **3** | 位时钟 64kHz~2.048MHz。频率 = 帧长 × 8kHz              |
|  8  | MCLKR/PDN |  接收  | **4** | 主时钟 / PDN=高电平时芯片进入掉电模式                   |
|  9  | MCLKX     |  发送  | **45** | 主时钟，A-law TP3057 需 2.048MHz                        |
|  10  | BCLKX     |  发送  | **44** | 位时钟 64kHz~2.048MHz，上升沿将数据移出 DX              |
|  12  | FSX       |  发送  | **47** | 8kHz 帧同步。FSX 为高期间，BCLKX 的上升沿将数据移出 DX  |

### 数字输出 (2)

| 引脚 | 信号 |    输出类型    |   ATE通道   | 说明                                            |
| :--: | ---- | :------------: | :----------: | ----------------------------------------------- |
|  11  | DX   | **三态** | **46** | 发送 PCM 数据出。仅编码时隙内驱动，其余时间高阻 |
|  13  | TSX  | **开漏** | **48** | 发送时隙指示。编码时隙内拉低，需外部上拉电阻    |

> **注意**：DX 和 TSX 均为输出类型，不应参与 IIL/IIH（输入漏电流）测试，应使用 IOZ（三态/高阻漏电流）测试方法。

### 模拟输入 (2)

| 引脚 | 信号   | 说明                                                  |
| :--: | ------ | ----------------------------------------------------- |
|  15  | VFXI− | 发送输入运放反相端。与 VFXI+ 构成差分输入             |
|  16  | VFXI+  | 发送输入运放同相端。音频信号由此进入 CODEC 的编码链路 |

### 模拟输出 (2)

| 引脚 | 信号 | 说明                                                          |
| :--: | ---- | ------------------------------------------------------------- |
|  3  | VFRO | 接收端音频输出。经 DAC + 5 阶低通滤波 + 功率放大，单位增益    |
|  14  | GSX  | 发送输入运放输出。外接 R1/R2 至 VFXI− 设定增益：Gain = R1/R2 |

## 收发信号链路

### 发送链路 (Tx: Analog → Digital)

```
VFXI+(16) ─┐
            ├─ 输入运放 ─→ GSX(14) ──→ BPF ──→ A-law 编码 ──→ DX(11)
VFXI−(15) ─┘                                              ↑
                                                    FSX(12) 触发
                                                    BCLKX(10) 移位
                                                    MCLKX(9) 主频
```

### 接收链路 (Rx: Digital → Analog)

```
DR(6) ──→ A-law 解码 ──→ 5阶 LPF ──→ 功率放大 ──→ VFRO(3)
   ↑                                       负载: 600Ω
FSR(5) 触发
BCLKR(7) 移位
MCLKR(8) 主频
```

## 关键参数速查

| 参数           | 值                                  |
| -------------- | ----------------------------------- |
| 正电源 VCC     | +5V ±5%                            |
| 负电源 VBB     | −5V ±5%                           |
| 主时钟 (A-law) | 2.048 MHz                           |
| 帧同步         | 8 kHz                               |
| 帧长           | 256 bit                             |
| 编码格式       | G.711 A-law + 偶数位反转 (XOR 0x55) |
| 满量程峰值     | 2.492V                              |
| 0 dBm0         | 1.2276 Vrms (600Ω)                 |
| 掉电功耗       | 典型 3 mW                           |
| 工作功耗       | 典型 50 mW                          |
